# Advanced Network Technologies
- This repository contains GNS3 portable projects.
- The projects are on Cisco equipment in IOS enviroment.
- A university project.

## Requirements
- [GNS3](https://www.gns3.com/)
- Router Used [Cisco 3725](https://www.gns3.com/marketplace/appliances/cisco-3725)
- Instructions for installation [here](https://docs.gns3.com/docs/using-gns3/beginners/import-gns3-appliance/)

## Project Explanation
### 1. Static Routes
Setting up routers and configuring static routes.

### 2. OSPF Single Area
Configuring OSPF on a single area.

### 3. OSPF Multiple Areas
Configuring OSPF on multiple areas.

### 4. OSPF Multiple Autonomous Systems
Configuring OSPF on multiple autonomous systems.

### 5. BGP with Network Statement
BGP with the use of network statement for advertising routes.

### 6. BGP with IGP
BGP with the use of IGP for advertising routes.

### 7. BGP with IGP
BGP with the use of AS-PATH.
